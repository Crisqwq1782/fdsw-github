Acerca de mi

Soy un estudiante de desarrollo Full Stack dentro de la academia Desafío Latam y soy un apasionado por transformar ideas conceptuales en soluciones web robustas y escalables. Mi experiencia abarca el manejo de aplicaciones como VScode, Git. HTML y CSS junto a frameworks como Bootstrap. Me considero una persona muy estudiosa que le encanta estar al dia con las ultimas novedades en la industria TI y además una de las cosas que mas disfruto es poner en practica mis conocimientos y así un dia garantizar al cliente la maxima satisfacción.

CSS

Manejo de CSS para dar estilo y formato a las páginas web, creando diseños atractivos y responsivos que mejoran la experiencia del usuario,la tipografía y los colores haciendo la pagina llamativa para el usuario promedio .

HTML

Uso de HTML para estructurar y organizar el contenido de las páginas web, creando una base sólida que facilita la navegación y mejora la accesibilidad para los usuarios y motores de búsqueda respetando las buenas practicas (SEO).

Bootstrap

Experiencia en el uso de Bootstrap para desarrollar sitios web responsivos y adaptativos, aprovechando su sistema de rejilla y componentes predefinidos para acelerar el proceso de desarrollo y garantizar una apariencia profesional en diversos dispositivos.

Mis trabajos

Cuppon Page

En este proyecto se desarrollo una pagina web para una empresa de cuppon, la cual ofrece descuentos en diferentes productos y servicios. La pagina web cuenta con un diseño atractivo y funcional, permitiendo a los usuarios navegar fácilmente y encontrar las mejores ofertas disponibles.

Menú Restaurante

Landpage desarrollado para un restaurante ficticio, en el cual se muestra su menú de platos y bebidas. El diseño es atractivo y funcional, permitiendo a los usuarios navegar fácilmente y encontrar la información que necesitan sobre los productos ofrecidos por el restaurante.

Portafolio con CV y Formulario.
