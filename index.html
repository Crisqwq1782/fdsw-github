Acerca de mí

Soy un estudiante de desarrollo Full Stack dentro de la academia Desafío Latam y soy un apasionado por transformar ideas conceptuales en soluciones web robustas y escalables. Mi experiencia abarca el manejo de aplicaciones como VS Code, Git, HTML y CSS junto a frameworks como Bootstrap. Me considero una persona muy estudiosa a la que le encanta estar al día con las últimas novedades en la industria TI. Una de las cosas que más disfruto es poner en práctica mis conocimientos para garantizar al cliente la máxima satisfacción.

CSS

Manejo de CSS para dar estilo y formato a las páginas web, creando diseños atractivos y responsivos que mejoran la experiencia del usuario, la tipografía y los colores.

HTML

Uso de HTML para estructurar y organizar el contenido, creando una base sólida que facilita la navegación y mejora la accesibilidad (SEO).

Bootstrap

Experiencia en el uso de Bootstrap para desarrollar sitios web responsivos y adaptativos, aprovechando su sistema de rejilla para acelerar el desarrollo.
